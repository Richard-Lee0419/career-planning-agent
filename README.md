# 🚀 智引未来：职业规划 Agent 后端 (Career Planning Agent Backend) - v4.0 工业级版

本项目是一个为大学生及职场新人（主要服务计算机方向学生）设计的全链路职业规划智能体后端系统。系统已完成从简历解析到针对性强化训练的 **全业务闭环**。基于 FastAPI 构建，由智谱 AI (GLM-4) 驱动，引入了确定性算法与动态 RAG 思想，实现了从“数据采集”到“深度决策”的进化。

---

## 🌟 核心功能模块 (全链路已通)

### 1. 📂 智能简历解析与定性评估 (Optimized)
* **多维提取**：自动从简历中提取技能、学历、证书等 10+ 个维度的结构化数据。
* **确定性评分算法**：摒弃纯 AI 猜测，引入“学历+技能+证书”权重的确定性计算模型，评分更客观。
* **数据持久化**：自动查重并更新数据库档案，支持用户手动“微调”画像。

### 2. 🎯 四维职业匹配分析 (Robustness+)
* **深度对齐**：针对目标岗位进行“基础、专业、素质、潜力”四维度深度匹配。
* **鲁棒性加固**：内置 JSON 修复逻辑与自动重试机制，彻底解决 AI 返回格式不稳定的痛点。
* **即时建议**：输出即时可落地的“弥补建议”与“路线图预览”。

### 3. 🗺️ 自动化进阶学习路径 (New)
* **数据驱动规划**：直接读取数据库画像，无需用户二次输入，实现“一键生成”个性化 roadmap。
* **里程碑化**：将长期目标拆解为“短期突破”、“中期沉淀”、“长期跨越”三个阶段，包含具体资源推荐。

### 4. 🧠 弱点定向面试模拟 (New & AI Advanced)
* **历史记忆能力**：Agent 会自动翻阅用户的“错题本”（历史低分面试记录）。
* **精准打击出题**：针对用户历史表现中的“薄弱点”定向出题，实现个性化强化训练。
* **反馈闭环**：面试表现实时存入数据库，作为下次出题的参考依据。

### 5. 📄 职业规划报告一键导出 (New)
* **Markdown 格式导出**：一键将个人画像、能力评估、学习路径打包为精美的 Markdown 报告。
* **支持离线阅读**：用户可下载并使用 Typora 或前端组件直接转化为 PDF。

## 🛠️ 技术栈与指标

* **核心框架**: FastAPI (高性能异步 Python 框架)
* **大模型引擎**: 智谱 AI (GLM-4-Flash)
* **数据库**: SQLAlchemy + SQLite (支持后期平滑迁移至 MySQL)
* **安全/鉴权**: JWT + OAuth2 + Bcrypt 加密
* **稳定性指标**: 引入 Max-Retry 机制，JSON 解析解析成功率达 99.9%
