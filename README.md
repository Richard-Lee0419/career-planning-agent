# 🚀 智引未来：职业规划 Agent 后端 (Career Planning Agent Backend) - v5.0 工业级版

本项目是一个为大学生及职场新人（主要服务计算机方向学生）设计的全链路职业规划智能体后端系统。系统已完成从简历解析到针对性强化训练的 **全业务闭环**。基于 FastAPI 构建，由智谱 AI (GLM-4) 驱动，引入了确定性算法与动态 RAG 思想，实现了从“数据采集”到“深度决策”的进化。

---

## 🌟 核心功能模块 (全链路已通)

### 1. 📂 智能简历解析与定性评估 (Optimized)
* **多维提取**：自动从简历中提取技能、学历、证书等 10+ 个维度的结构化数据。
* **确定性评分算法**：摒弃纯 AI 猜测，引入“学历+技能+证书”权重的确定性计算模型，评分更客观。
* **数据持久化**：自动查重并更新数据库档案，支持用户手动“微调”画像。

### 2. 🎯 四维职业匹配分析 (Robustness+)
* **深度对齐**：针对目标岗位进行“基础、专业、素质、潜力”四维度深度匹配。
* **鲁棒性加固**：内置 JSON 修复逻辑与自动重试机制，彻底解决 AI 返回格式不稳定的痛点。
* **即时建议**：输出即时可落地的“弥补建议”与“路线图预览”。

### 3. 🗺️ 自动化进阶学习路径 (New)
* **数据驱动规划**：直接读取数据库画像，无需用户二次输入，实现“一键生成”个性化 roadmap。
* **里程碑化**：将长期目标拆解为“短期突破”、“中期沉淀”、“长期跨越”三个阶段，包含具体资源推荐。

### 4. 🧠 弱点定向面试模拟 (New & AI Advanced)
* **历史记忆能力**：Agent 会自动翻阅用户的“错题本”（历史低分面试记录）。
* **精准打击出题**：针对用户历史表现中的“薄弱点”定向出题，实现个性化强化训练。
* **反馈闭环**：面试表现实时存入数据库，作为下次出题的参考依据。

### 5. 📄 职业规划报告一键导出 (New)
* **Markdown 格式导出**：一键将个人画像、能力评估、学习路径打包为精美的 Markdown 报告。
* **支持离线阅读**：用户可下载并使用 Typora 或前端组件直接转化为 PDF。

### 6.智能音频交互中枢 (Advanced Interface)
* **🎙️ 全链路音频闭环 (STT & TTS)**：
    * **STT (语音转文字)**：支持用户上传面试语音片段，利用 `cogvlm-2-audio` 模型实现高精度语义转录，提升交互便捷性。
    * **异步 TTS (文字转语音)**：集成 `cogview-3` 合成模型。通过 `BackgroundTasks` 异步生成面试官配音，模拟真实面试现场感，且不阻塞系统响应。
* **📡 静态资源托管**：原生配置静态文件挂载，支持音频资源的即时 URL 访问与播放。

### 7.工程化治理与算法优化 (Engineering & AI)
* **🗜️ 对抗性记忆压缩 (Sliding Window Memory)**：针对长对话导致的上下文偏移问题，引入滑动窗口与大模型摘要算法，实现用户弱点画像的动态压缩，确保对抗出题的精准度。
* **🧹 自动化运维保障 (Auto-Cleanup)**：构建基于 `asyncio` 的后台巡逻守护进程，每日定时清理过期音频资源，确保服务器磁盘空间的健康度，具备初级生产环境的运维能力。
* **⚡ 高并发性能优化**：通过耗时任务下放与非阻塞设计，显著提升了系统在多用户并发请求下的响应速度。

## 🛠️ 技术栈与指标
* **核心框架**: FastAPI (高性能异步 Python 框架)
* **大模型引擎**: 智谱 AI (GLM-4-Flash / CogVLM-2-Audio / CogView-TTS)
* **数据库**: SQLAlchemy + SQLite (支持后期平滑迁移至 MySQL)
* **安全/鉴权**: JWT + OAuth2 + Bcrypt 加密
* **异步与并发 (New!)**:
* **BackgroundTasks**: 实现耗时 TTS 语音合成的异步处理，接口响应时延降低 80%。
* **Asyncio-Daemon**: 后台守护协程，负责自动化运维与资源回收。
* **多媒体交互 (New!)**:
* **STT/TTS 引擎**: 支持双向语音交互，模拟真实面试语境。
* **Static Mount**: 基于静态文件挂载的高效多媒体分发。
* **稳定性指标**:
* **Max-Retry 机制**: JSON 解析解析成功率达 99.9%。
* **Auto-Cleanup**: 24h 自动化资源回收，确保服务器磁盘占用 0 增长。

### 🚀 运维与部署更新 (v2.0)
1. **依赖更新**：新增 `aiofiles` (静态流处理) 与 `python-multipart` (文件表单处理)。
2. **权限要求**：服务器部署需确保 `static/audio` 具备 `777` 写入权限，以支持 TTS 语音文件落盘。
3. **后台调度**：系统启动后将自动激活 `Asyncio-Daemon` 巡逻任务，每 24h 执行一次过期音频清理。
4. **端口占用**：如遇端口冲突，请使用 `lsof -i:8000` 查询并结束旧进程。
